# 📝 Todo App

> Minimalist task manager with drag-and-drop sorting and smooth animations

## Stack
![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)
![Vite](https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=FFFFFF)
![dnd-kit](https://img.shields.io/badge/dnd--kit-FF4B4B?style=for-the-badge)

## Features
- Create, edit, delete & toggle tasks
- Vertical drag-and-drop reordering
- Fully responsive layout
- Task filtering
- Notifications

## Quick Start
```
git clone https://github.com/she1nXgod/react-todo-app.git
cd react-todo-app
npm ci
npm run dev
```
